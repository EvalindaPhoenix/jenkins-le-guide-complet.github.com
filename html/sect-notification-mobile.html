8.11. Notifications vers mobiles

Si votre serveur Jenkins est visible sur Internet (même si vous avez mis en place une authentification sur votre serveur Jenkins), vous pouvez aussi surveiller vos builds via votre appareil mobile iPhone ou Android. L'application gratuite Hudson Helper (voir Figure 8.25, “Utiliser l'application iPhone Hudson Helper”) par exemple, vous permet de lister vos tâches de builds actuelles (soit l'ensemble des tâches de builds sur le serveur, ou seulement les tâches de build d'une certaine vue). Vous pouvez également afficher les détails d'une tâche de build particulière, y compris son statut actuel, les tests en échec et le temps de build, et même démarrer et arrêter les builds.

Figure 8.25. Utiliser l'application iPhone Hudson Helper


Pour les téléphones Android, vous pouvez également installer le widget Hudson Mood qui fournit également des mises à jour et des alertes sur les échecs de build.

Notez que ces applications mobiles s'appuient sur une connexion de données, de sorte qu'ils travaillent généralement bien localement, mais vous ne devriez pas compter sur eux si le développeur est à l'extérieur.
